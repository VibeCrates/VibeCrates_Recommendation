## 1. 실험 개요

  목적:
    학습된 DualEncoderModel이 영어 쿼리에 대해 어떤 품질의 추천을 내놓는지
    스타일·도메인·쿼리 단위로 정량 평가한다.

  쿼리 구성:
    - 4가지 스타일 × 4쌍 = 16개 영어 쿼리
    - 스타일: philosophical(P), poet(T), atmosphere(A), direct(D)
    - 각 쿼리 × 4 도메인 필터(movie/music/book/all) × top-5 = 최대 320행

  평가 방법:
    - item_id 기반으로 data/train_dataset.csv의 content_text를 읽어
      쿼리와의 관련성을 3단계로 평가
    - o: 쿼리에 부합하는 최상 추천
    - a: 부분적으로 관련 있는 허용 가능한 추천
    - x: 쿼리와 무관한 실패 추천
    - 155행은 사람(사용자)이 직접 평가, 165행은 Claude가 content_text 기반 평가

  점수 환산:
    o=2, a=1, x=0 으로 환산하여 avg_score를 산출

    추상 이미지 vs 감정 키워드 간 성능 분기가 핵심이다.

## 3-3. Rank 순위가 품질을 보장하지 않는다

  ┌────────┬───────────┐
  │ rank   │ avg_score │
  ├────────┼───────────┤
  │   1    │   1.312   │
  │   2    │   1.094   │
  │   3    │   1.250   │  ← rank 2보다 높음 (역전)
  │   4    │   0.984   │
  │   5    │   1.094   │  ← rank 4보다 높음 (역전)
  └────────┴───────────┘

  rank 1 > rank 2 > rank 3 순서로 단조 감소해야 하는데
  rank 3 > rank 2, rank 5 = rank 2 역전이 발생한다.

  코사인 유사도 기반 top-k 정렬이 실제 품질 순서와 일치하지 않는다.
  표현 공간이 아직 충분히 정제되지 않아 유사도 값 자체의 신뢰도가 낮은 상태.

## 4. 총평

  모델의 현재 추천 품질은 쿼리 스타일에 따라 크게 갈린다.
  추상적 개념을 표현하는 철학적 쿼리에서는 준수한 성능을 보이지만,
  감각 이미지나 분위기 기반 쿼리에서는 표현 공간의 정렬이 부족하다.

  특히 두 가지 구조적 문제가 확인된다:
    1) 코사인 유사도 순위가 실제 품질 순위와 일치하지 않아
       top-1 추천의 신뢰도가 낮다.
    2) 키워드 오정렬(D1_EN 사례)이 발생할 경우 전체 추천이 붕괴된다.


## 5. 개선 방향

  [우선순위 높음]

  5-1. Hard Negative Mining
    - D1_EN 사례처럼 "Action" → 영화 제작 도서, "space" → 추상 공간 오정렬
    - 현재 배치 내 랜덤 negative는 너무 쉬워서 경계 케이스를 학습 못 함
    - In-batch hard negative (가장 높은 cosine 쌍을 negative로 선택)로
      표현 공간 경계를 세밀하게 조정 필요

  5-2. Poet/Atmosphere 스타일 쿼리 학습 데이터 증강
    - 현재 학습 쿼리가 직접적(Direct) 특성 위주로 구성된 것으로 추정
    - "A voice whispering under moonlight" 같은 감각 이미지 쿼리를
      학습 데이터에 추가하면 T/A 스타일 성능 향상 기대

  5-3. Stage 2 Epoch 확장
    - Knowledge Distillation이 더 진행될수록 CLIP 교사 모델의
      강한 이미지-텍스트 정렬이 학생 모델로 이전됨
    - rank 순위 신뢰도 향상에 간접적으로 기여할 수 있음

  [우선순위 중간]

  5-4. Music 도메인 임베딩 품질 개선
    - 가사 텍스트가 있어도 쿼리와 맞지 않을 경우 크게 어긋남
    - 가사 대신 장르·분위기·감정 태그를 보강한 content_text 재구성 검토

  5-5. Rank 순위 보정
    - 현재 top-5를 코사인 유사도 순으로 그대로 반환
    - 표현 공간이 개선될 때까지 임시방편으로 diversity-aware reranking 검토
